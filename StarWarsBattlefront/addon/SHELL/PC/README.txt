The shell.lvl shipped with the ALT Addon System is for the STEAM version of BF1 (6/24/2020).
If you have a different version that you want to use instead, place the SHELL.LVL here and use 
LVLTool to replace the shell_interface lua script:
   1. Browse to AltAddonSystem\StarWarsBattlefront\addon\bin
   2. Double click 'LVLTool.exe'
   3. Select "C:\BFBuilder" from the top dropdown
   4. File -> Enter ModTools Dir -> Enter the path to your "BFBuilder" folder 
   5. Drag the target 'shell.lvl' into the file text box (at the top of LVLTool)
   6. Select 'shell_interface' in the list box
   7. Press 'Replace item' button
   8. Select the included 'addon\SHELL\PC\shell_interface.lua' file
   9. Press 'Save lvl file' button (left bottom of LVLTool Form)
   10. Select the "addon\SHELL\PC\shell.lvl" file and confirm overwrite.
(Refer to video mentioned in LVLTool Readme if neded https://github.com/BAD-AL/LVLTool )
